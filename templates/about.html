About the Project

This project aims to predict housing prices in Delhi using a range of machine learning models. The dataset used for training the model is sourced from Kaggle, providing a comprehensive set of features to accurately predict property values.

Machine Learning Models

Various machine learning models were evaluated to find the best predictor for housing prices:

  • Random Forest:Achieved the highest accuracy of 93% and was selected for its superior performance.
  • Decision Tree: Used a tree-like model of decisions to predict prices.
  • Gradient Boosting: An ensemble method that builds models sequentially to correct errors.
  • Linear Regression: Provided a simple linear approach to predicting prices.
  • XGBRegressor:
  • CatBoosting Regressor: Effective for categorical features and provided robust predictions.
  • AdaBoost Regressor: Boosted weak learners to improve accuracy.
